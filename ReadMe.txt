Sindy::SQLite是一个访问sqlite3数据库的C++操作类

Sindy::Range2d及配套的class是一个提升大量二维实体求交效率的C++类，只要这些实体能求得其最大包络，
即可以求得这样一个“外接矩形”（可以用左下、右上两个个点描述），该图形的任意部分都在矩形内。
在TestRangeBound.cpp中有相关的测试用例，该方法在ObjectARX（CAD二次开发库）中很容易实现，且效果明显。
